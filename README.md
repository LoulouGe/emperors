# Le Jeu des Empereurs Romains

Un jeu interactif pour apprendre les dates de regne des empereurs romains, d'Auguste a Severe Alexandre.

## Comment jouer

1. Ouvrir `index.html` dans un navigateur
2. Cliquer sur **Jeu des Empereurs Romains** pour commencer
3. Cliquer sur **Jouer**
4. Pour chaque empereur, choisir les bonnes dates de regne parmi les 4 propositions
5. A la fin des 26 questions, un score final s'affiche

## Structure du projet

```
emperors/
  index.html      <- le site (HTML + CSS + JavaScript)
  images/         <- les portraits des empereurs
  sounds/         <- les fichiers audio (questions et feedback)
```

## Technologies

- HTML / CSS / JavaScript (vanilla, pas de framework)
- Fichiers audio MP3 pour les questions et le feedback
- Images JPG/WebP pour les portraits
